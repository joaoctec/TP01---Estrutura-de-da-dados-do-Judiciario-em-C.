N1 de Estrutura de Dados - Código de Análise de Casos Judiciários do TJDFT em C

Este projeto é um programa em C que analisa um grande volume de dados de processos judiciais do Tribunal de Justiça do Distrito Federal e Territórios (TJDFT) a partir de um arquivo CSV. 
Ele faz análises quantitativas, gera o relatório de casos por categoria e gera estatísticas de  previsão do tempo de resolução, assim ajudando a extrair informações importantes para estudo ou pesquisa.
